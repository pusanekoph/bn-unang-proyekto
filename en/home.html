Basic Help

       To advance through the game, left-click or press the space or enter keys. When at a menu, left-click to make a choice, or use the arrow keys to select a choice and enter to activate it.

Main Menu

       After launching the program, you will be brought to the main menu. The main menu should be fairly self-explanatory, giving you the ability to:

  • Start:

    •        Starts a new game.

  • Load:

    •        Load a saved game.

  • Option:

    •        Configure game options.

  • Quit:

    •        Quit entirely.

In-Game

       After loading a game, the following controls become active:

  • Left Click, Space, Enter:

    •        Advance to the next screen of dialogue.

  • Left Click, Enter:

    •        Activate a menu choice or button.

  • Right Click, Escape:

    •        Brings you to the game menu.

  • Center Click, "H":

    •        Hides transient windows, allowing one to see the pictures unobstructed.

  • Mouse Wheel Up, Page Up:

    •        Rolls back to a prior point in the game. This allows you to review previously-read text, and even change mistaken choices.

  • Mouse Wheel Down, Page Down:

    •        Rolls forward when viewing text that was shown using rollback.

  • Ctrl:

    •        Skips for as long as the Ctrl key is held down.

  • Tab:

    •        Toggles skipping until tab is pressed again.

  • "F":

    •        Toggles fullscreen mode.

Game Menu

       The game menu gives the following options:

  • Return:

    •        Returns to the game in progress.

  • Skip:

    •        Returns to the game, skipping.

  • Option:

    •        Allows one to adjust preferences.

  • Save:

    •        Save your current game position.

  • Load:

    •        Load a saved game position.

  • Main Menu:

    •        Brings you back to the main menu.

  • Quit:

    •        Exit game
